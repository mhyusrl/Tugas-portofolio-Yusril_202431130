**Struktur Halaman Website**
1. Halaman Home / Beranda (index.html) Halaman utama yang berfungsi sebagai pengenalan diri siswa. Bagian-bagian yang harus ada:
   - Header & Navigasi;Navigasi utama yang konsisten di seluruh halaman (Home, Portofolio, dan Kontak).
   - Hero Section: Menampilkan foto diri dan deskripsi singkat sebagai pengantar profil pribadi.
   - Tentang Saya: Berisi informasi mengenai latar belakang, pendidikan, minat, dan kepribadian.
   - Keterampilan/Keahlian: Menampilkan kemampuan utama dalam bentuk ikon atau bilah kemajuan (progress bar).
   - Footer: Harus menyertakan tautan resmi: https://itpln.ac.id

2. Halaman Data / Portofolio (portfolio.html atau data.html) Halaman ini menampilkan daftar proyek yang pernah dibuat atau dirancang (boleh data dummy). Ketentuan:
   - Memiliki judul halaman yang jelas.
   - Menampilkan tabel data proyek dengan kolom:
   - Nama Proyek Tahun Pembuatan Teknologi yang Digunakan Status (Selesai/Belum) Tautan Demo (opsional)
   - Tabel harus responsif dan tidak rusak ketika diakses melalui ponsel.
   - nimal terdapat 5 data proyek.

3. Halaman Kontak / Formulir (contact.html atau form.html) Halaman ini berisi materi interaktif untuk berkomunikasi dengan pengunjung. Mahasiswa dapat memilih salah satu
   jenis formulir berikut:
   - Formulir Permintaan Proyek
   - Formulir Konsultasi Teknologi
   - Formulir Tinjauan Portofolio
   - Elemen wajib dalam form:
   - Masukkan teks: Nama dan Email Dropdown: Jenis layanan atau topik Textarea: Pesan atau deskripsi proyek Kotak centang atau tombol radio Tombol kirim
   - Bentuknya harus ramah pengguna (user-friendly) dan tetap responsif di berbagai perangkat.

4. Persyaratan Penerapan Teknis portofolio hasi pengerjaan
   - Situs web diunggah ke repositori GitHub publik.
   - GitHub Pages harus diaktifkan agar website dapat diakses secara online.
   - Setiap halaman wajib menyertakan tautan ke https://itpln.ac.id di bagian footer.
   - Navigasi antar halaman harus saling terhubung.
   - Sertakan file README.md yang informatif, berisi penjelasan proyek, struktur halaman, serta tautan menuju website yang telah dipublikasikan.
